```html
<button popovertarget="my-popover">팝오버 열기</button>
<div popover id="my-popover">팝오버 열림!</div>
```
<div class="p-4 border">
  <button popovertarget="my-popover" class="border">팝오버 열기</button>
  <div popover id="my-popover" class="border">팝오버 열림!</div>
</div>
팝오버 태그

```html
<p>이건 <mark>마킹</mark> 입니다</p>
```
<div class="p-4 border">
  <p>이건 <mark>마킹</mark> 입니다</p>

텍스트를 하이라이팅 할 수 있다.

```html
<input type="text" list="movie-options" />
<datalist id="movie-options">
  <option value="30일"/>
  <option value="범죄도시3"/>
  <option value="엘리멘탈"/>
</datalist>
```
<div class="p-4 border">
  <input type="text" list="movie-options" class="border" />
  <datalist id="movie-options">
    <option value="30일"/>
    <option value="범죄도시3"/>
    <option value="엘리멘탈"/>
  </datalist>
</div>

순수 html만으로 autocomplete 기능을 사용 할 수 있다.

```html
<input type="date" />
<input type="week" />
<input type="month" />
<input type="time" />
```
<div class="p-4 border">
  <input type="date" class="border mb-2" /><br />
  <input type="week" class="border mb-2" /><br />
  <input type="month" class="border mb-2" /><br />
  <input type="time" class="border" />
</div>

input 태그의 type 설정 만으로도 날짜 데이터를 입력 받을 수 있다

```html
<details>
  <summary>클릭 전 볼 수 있는 영역</summary>
  <span>클릭 후에만 표시되는 영역</span>
</details>
```
<div class="p-4 border">
  <details>
    <summary>클릭 전 볼 수 있는 영역</summary>
    <span>클릭 후에만 표시되는 영역</span>
  </details>
</div>

아코디언 기능을 순수 html로 비슷하게 구현 할 수 있는 듯 하다.

```html
<select name="cars" id="cars">
  <optgroup label="현대차">
    <option value="sonata">쏘나타</option>
    <option value="avante">아반떼</option>
  </optgroup>
  <optgroup label="기아차">
    <option value="k5">K5</option>
    <option value="k3">K3</option>
  </optgroup>
</select>
```
<div class="p-4 border">
  <select name="cars" id="cars" class="border">
    <optgroup label="현대차">
      <option value="sonata">쏘나타</option>
      <option value="avante">아반떼</option>
    </optgroup>
    <optgroup label="기아차">
      <option value="k5">K5</option>
      <option value="k3">K3</option>
    </optgroup>
  </select>
</div>

`optgroup` 태그를 사용해 선택 옵션을 그룹으로 나눌 수 있다

```html
<dialog open>
  <p>다이얼로그</p>
  <form method="dialog">
    <button>확인</button>
  </form>
</dialog>
```
<div class="p-4 border">
  <dialog open class="relative px-10">
    <p>다이얼로그</p>
    <form method="dialog">
      <button class="border py-2 px-6 mt-4 bg-gray-500">확인</button>
    </form>
  </dialog>
</div>

dialog를 쉽게 사용가능하다

```html
<ruby>노바로그<rt>노바의 개발 블로그</rt></ruby>
```
<div class="p-4 border">
  <ruby>노바로그<rt>노바의 개발 블로그</rt></ruby>
</div>
해당문자의 설명을 윗 첨자로 표현
```html
<meter min="0" max="100" value="50" />
<progress min="0" max="100" value="50" />
```
<div class="p-4 border">
  <meter min="0" max="100" value="55" />
  <br />
  <progress min="0" max="100" value="50" />
</div>  

`meter` 와 `progress` 태그는 값을 게이지 형태로 표현한다

```html
<kbd>ALT</kbd>
```
<div class="p-4 border">
  <kbd>ALT</kbd>
</div>
키보드 입력을 나타낼 때 사용
```html
<attr title="별걸 다 줄이네">별다줄</attr>
```
<div class="p-4 border">
  <attr title="별걸 다 줄이네">별다줄</attr>
</div>  

축약형 단어를 쓸 때 사용
```html
<input type="email" id="email" />
```
<div class="p-4 border">
  <input type="email" id="email" class="border" />
</div>  

type과 id를 email로 주면 chrome 기준 이메일 자동완성 기능이 동작한다

```html
<input type="text" autofocus />
```
autofocus 속성을 주면 페이지가 로드될 때 자동으로 포커스를 갖는다

```html
<label for="name">닉네임 : </label>
<input type="text" id="name" />
```
<div class="p-4 border">
  <label for="name">닉네임 : </label>
  <input type="text" id="name" class="border" />
</div>

id를주고 label for과 매칭시키면 label을 클릭했을 때에도 input에 포커싱된다 

```html
<button type="reset">버튼</button>
```
reset type은 form 안의 양식을 초기화 시켜준다
