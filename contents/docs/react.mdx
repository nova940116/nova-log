const sampleComponent = () => {
  const basicCondition = flag && flag2 && !flag3

복잡한 조건부 렌더링 처리 기법
```jsx
const sampleComponent = () => {
  return isTrue && <p>True!</p> // isTrue ? <p>True!</p> : null 대체
}
```
단순 조건부 렌더링
```jsx
const pages = 5

{[...Array(pages)].map((_, i) =>
  <Link key={i} href={`/${i ? `?page=${i + 1}` : ''}`}>
    ...
  </Link>
)}
```
숫자 값을 배열로 렌더링 해야 할 때 `[...Array(number)]` Destructuring을 사용 할 수 있다.

```jsx title="Parent.jsx"
const Parent = () => {
  const receivedFromChild = (value) => {
    console.log(value) // hello
  }
  return <Child callingParent={receivedFromChild} />
}
```

```jsx title="Child.jsx"
const Child = ({ callingParent }) => {
  callingParent("hello")
  return <div>Child</div>
}
```
