## 1. structuredClone

원본에 영향을 주지 않고 객체를 등록/수정/삭제 해야 할 때, `structuredClone` 함수를 사용 할 수 있습니다.
객체 복사는 spread operator를 사용하면 되는거 아니냐구요? 물론 가능하지만 다음과 같은 문제가 발생 할 수 있습니다.

```Javascript
const original = {
  someProp: 'value',
  anotherProp: {
      withAnotherProp: true
  }
}

const shallowCopy = { ...original }

shallowCopy.newProp = 'new value'
shallowCopy.anotherProp.newProp = 'new value'

console.log(original.newProp) // undefined
console.log(original.anotherProp.newProp) // 'new value'
```
위의 코드와 같이 spread operator를 통해 복사했더라도, 중첩된
