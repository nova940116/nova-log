```js
const pages = 5

{[...Array(pages)].map((_, i) =>
  <Link key={i} href={`/${i ? `?page=${i + 1}` : ''}`}>
    ...
  </Link>
)}
```
숫자 값을 배열로 렌더링 해야 할 때 `[...Array(number)]` Destructuring을 사용 할 수 있다.

```tsx title="Parent.tsx"
const Parent = () => {
  const receivedFromChild = (value) => {
    console.log(value)
  }
  return <Child callingParent={receivedFromChild}></Child>
}
```

```tsx title="Child.tsx"
const Child = ({ callingParent }) => {
  callingParent("hello")
  return <div>Child</div>
}
```
