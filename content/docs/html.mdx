```html
<dialog open>
  <p>Greetings, one and all!</p>
  <form method="dialog">
    <button>OK</button>
  </form>
</dialog>
```
<dialog open>
  <p>Greetings, one and all!</p>
  <form method="dialog">
    <button>OK</button>
  </form>
</dialog>

dialog를 쉽게 사용가능하다

```html
<ruby>노바로그<rt>노바의 개발 블로그</rt></ruby>
```
<ruby>노바로그<rt>노바의 개발 블로그</rt></ruby>

해당문자의 설명을 윗 첨자로 표현
```html
<meter min="0" max="10" value="5" />
```
<meter min="0" max="10" value="5" />

값을 게이지 형태로 표현

```html
<kbd>ALT</kbd>
```
<kbd>ALT</kbd>

키보드 입력을 나타낼 때 사용
```html
<attr title="별걸 다 줄이네">별다줄</attr>
```
<attr title="별걸 다 줄이네">별다줄</attr>

축약형 단어를 쓸 때 사용
```html
<input type="email" id="email" />
```
<input type="email" id="email" style={{ border: '1px solid #dbdbdb' }} />

type과 id를 email로 주면 chrome 기준 이메일 자동완성 기능이 동작한다

```html
<input type="text" autofocus />
```
autofocus 속성을 주면 페이지가 로드될 때 자동으로 포커스를 갖는다

```html
<label for="name">닉네임 :</label>
<input type="text" id="name" />
```
<label for="name">닉네임 :</label>
<input type="text" id="name" style={{ border: '1px solid #dbdbdb' }} />

id를주고 label for과 매칭시키면 label을 클릭했을 때에도 input에 포커싱된다 

```html
<button type="reset">버튼</button>
```
reset type은 form 안의 양식을 초기화 시켜준다
